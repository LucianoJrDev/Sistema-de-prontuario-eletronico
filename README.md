## Sistema de Prontuário Eletrônico (Healthcare System)

O que é: Um sistema para clínicas/hospitais gerenciarem pacientes.

Funcionalidades:

Cadastro de pacientes e médicos

Histórico médico completo

Agendamento de consultas

Upload de exames (PDF/imagem)

Controle de acesso por perfil (admin/médico)

Diferenciais:

LGPD (simulação de proteção de dados sensíveis)

Logs de acesso (quem visualizou o quê)

Stack sugerida: Node.js + PostgreSQL + React + AWS S3 (ou similar)
